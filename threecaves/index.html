Three caves

A Minecraft data pack that adds three new cave biomes

Fungus caves

Caves overgrown with mushrooms and glow lichen that
generate beneath deep oceans and mushroom islands

Hot caves

Caves with lava and obsidian, magma block flats, and
basalt formations near the bottom of the world

Ice caves

Frozen caves covered in ice and snow with increased skeleton rates

Icicles are dripstone retextured with a resource pack; looks like this otherwise
Alternatively, use a version of the data pack without pointed dripstone in ice caves
If you know more about biome-specific textures, please hit me up so I can improve this!

Technical details

This pack overwrites data/minecraft/dimension/overworld.json. This may conflict with other world generation datapacks active in a world. To make Three Caves compatible with another pack, use the following info for Three Caves biome generation in the overworld.json you want your world to actually use:

{
  "biome": "three_caves:fungus_cave",
  "parameters": {
    "continentalness": [-1.3,-1.2],
    "depth": [0.7,0.9],
    "erosion": [-1,1],
    "humidity": [-1,1],
    "offset": 0,
    "temperature": [-1,1],
    "weirdness": [-1,1]
  }
}, {
  "biome": "three_caves:ice_cave",
  "parameters": {
    "continentalness": [-1,1],
    "depth": [0.3,0.9],
    "erosion": [0.3,1],
    "humidity": [0.6,1],
    "offset": 0,
    "temperature": [-1,-0.5],
    "weirdness": [0,1]
  }
}, {
  "biome": "three_caves:hot_cave",
  "parameters": {
    "continentalness": [-1,1],
    "depth": 1,
    "erosion": [0.3,1],
    "humidity": [0,0.2],
    "offset": 0,
    "temperature": [0,1],
    "weirdness": [0,1]
  }
}

The overworld.json in this data pack is just the vanilla overworld with these three biomes added.
